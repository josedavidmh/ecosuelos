Desarrollo de sistema de información para la clasificación de las tierras por su capacidad de uso - Ecosuelos

Resumen
Básicamente la idea es crear una aplicación de software que realice cálculos para determinar la capacidad de uso de los suelos de acuerdo con la metodología IGAC para la clasificación de las tierras por su capacidad de uso y de esta manera poder analizar sus características y propiedades.
Palabras Clave: Metodología IGAC, Suelos, Clasificación, Capacidad, Uso.
